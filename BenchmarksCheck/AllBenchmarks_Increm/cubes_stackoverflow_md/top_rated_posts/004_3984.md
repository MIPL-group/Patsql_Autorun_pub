# solution

```sql
SELECT a.id, a.rev, a.content
FROM i004 AS a
JOIN (
    SELECT id, MAX(rev) AS maxrev
    FROM i004
    GROUP BY id
) b ON a.id = b.id AND a.rev = b.maxrev
```
