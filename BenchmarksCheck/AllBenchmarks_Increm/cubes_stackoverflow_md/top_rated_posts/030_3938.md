# solution

```sql
SELECT a.* FROM i030 AS a
  LEFT JOIN i030 AS a2
    ON a.groupname = a2.groupname AND a.age <= a2.age
GROUP BY a.person
HAVING COUNT(*) >= 2
ORDER BY a.groupname, a.age DESC;
```
