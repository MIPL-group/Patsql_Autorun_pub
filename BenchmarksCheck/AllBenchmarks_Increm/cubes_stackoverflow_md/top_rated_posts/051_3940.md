# solution

```sql
SELECT t1.*
FROM i051 AS t1
         JOIN
     (
         SELECT category, MAX(date) AS MAXDATE
         FROM i051
         GROUP BY category
     ) AS t2
     ON T1.category = t2.category
         AND t1.date = t2.MAXDATE
```
