# solution

```sql
WITH NumberedMyTable AS
         (
             SELECT Id, Value, ROW_NUMBER() OVER (ORDER BY Id) AS RowNumber
             FROM i025
         )
SELECT Id, Value
FROM NumberedMyTable
WHERE RowNumber BETWEEN 3 AND 4
```
