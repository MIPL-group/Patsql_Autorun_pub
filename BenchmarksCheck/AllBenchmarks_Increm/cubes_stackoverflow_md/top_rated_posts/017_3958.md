# solution

```sql
SELECT t.*
FROM i017 AS t
         JOIN (SELECT MIN(ID) as MinID
                     FROM i017
                     WHERE SKU LIKE 'FOO-%'
                     GROUP BY PRODUCT
) dt ON t.ID = dt.MinID

```
