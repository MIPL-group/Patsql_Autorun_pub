# solution

```sql
SELECT t.Train, t.Dest, r.MaxTime
FROM (
      SELECT Train, MAX(Time) as MaxTime
      FROM i034
      GROUP BY Train
) r
INNER JOIN i034 t
ON t.Train = r.Train AND t.Time = r.MaxTime
```
