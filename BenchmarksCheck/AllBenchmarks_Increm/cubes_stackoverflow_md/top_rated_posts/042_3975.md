# solution

```sql
select name, Red, Green, Blue
from
(
  select color, name, value
  from yourtable
  unpivot
  (
    value for name in (Paul, John, Tim, Eric)
  ) unpiv
) src
pivot
(
  sum(value)
  for color in (Red, Green, Blue)
) piv
```
