# solution

```sql
select emp_id,emp_sal,emp_grp
from (
select e.*,
row_number() over(partition by emp_id
                  order by case when emp_grp = 'HMCPR' then 0 else 1 end,emp_sal) as rn
from "017" AS e
) t
where rn = 1;
```
