# solution

```sql
SELECT menu.name, sum(orderregel.quantity)    as quantity
  FROM "045_1" AS menu
  LEFT JOIN "045_2" orderregel
  on menu.menu_id = orderregel.menu_id
  group by menu.menu_id

```
