# solution

```sql
select t1.Employee, t1.mindt, t2.maxdt, t1.code
from
(select min(FromDate) mindt, Code, Employee
from "007"
group by Employee, Code) t1
join
(select max(ToDate) maxdt, Code, Employee
from "007"
group by Employee, Code) t2
on t1.code = t2.code
```
