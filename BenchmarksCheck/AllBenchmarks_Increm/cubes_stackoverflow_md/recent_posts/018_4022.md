# solution

```sql
SELECT t1.Course, Count(t2.OrderID)
FROM "018" AS t1
LEFT JOIN
 (
   SELECT CustomerID, Min(OrderID) AS OrderID
   FROM "018" AS t2
   WHERE Date='17.10.2016'
   GROUP BY CustomerID
 ) AS t2
ON t1.CustomerID = t2.CustomerID
AND t1.OrderID = t2.OrderID
WHERE Date='17.10.2016'
GROUP BY t1.Course

```
