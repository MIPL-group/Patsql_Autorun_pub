# solution

```sql
select tz.zone_id
     , tz.zone_name
     , max(tz.time_start)
from timezone tz 
where tz.time_start < strftime('%s', 'now')
group by tz.zone_id
       , tz.zone_name
order by tz.zone_id;
```
