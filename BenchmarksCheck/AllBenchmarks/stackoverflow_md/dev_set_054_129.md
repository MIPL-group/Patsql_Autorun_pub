# solution

```sql
SELECT  MaxT.Person,
        MaxT.Duration,
        T.Value,
        MIN(T.Uniq) AS Uniq
FROM    T
        INNER JOIN
        (   SELECT Person, SUM(`Time`) AS Duration, MAX(Value) AS Value
            FROM    T
            GROUP BY Person
        ) MaxT
            ON MaxT.Person = t.Person
            AND MaxT.Value = t.Value
GROUP BY MaxT.Person, MaxT.Duration, MaxT.Value;
```
