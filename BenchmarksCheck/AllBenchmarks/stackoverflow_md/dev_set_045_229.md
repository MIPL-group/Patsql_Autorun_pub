# solution

```sql
select t1.*
from tablename t1
    join (select BOM, AltBOM, MAX(ChangeNo) as ChangeNo
          from tablename
          group by BOM, AltBOM) t2
    on t1.BOM = t2.BOM and t1.AltBOM = t2.AltBOM and t1.ChangeNo = t2.ChangeNo
```
