# solution

```sql
SELECT T2.invoice_number,
       T2.invoice_date
FROM Financial_transactions AS T1
JOIN Invoices AS T2 ON T1.invoice_number = T2.invoice_number
GROUP BY T1.invoice_number
ORDER BY count(*) DESC
LIMIT 1
```
