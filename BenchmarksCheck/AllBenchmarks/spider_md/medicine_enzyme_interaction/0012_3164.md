# solution

```sql
SELECT T1.interaction_type
FROM medicine_enzyme_interaction AS T1
JOIN medicine AS T2 ON T1.medicine_id = T2.id
JOIN enzyme AS T3 ON T1.enzyme_id = T3.id
WHERE T3.name = 'ALA synthase'
  AND T2.name = 'Aripiprazole'
```
