# solution

```sql
SELECT T1.name
FROM browser AS T1
JOIN accelerator_compatible_browser AS T2 ON T1.id = T2.browser_id
JOIN web_client_accelerator AS T3 ON T2.accelerator_id = T3.id
WHERE T3.name = 'CProxy'
  AND T2.compatible_since_year > 1998
```
