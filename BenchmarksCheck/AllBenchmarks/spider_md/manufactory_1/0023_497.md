# solution

```sql
SELECT count(DISTINCT name)
FROM products
WHERE name NOT IN
    (SELECT T1.name
     FROM products AS T1
     JOIN manufacturers AS T2 ON T1.Manufacturer = T2.code
     WHERE T2.name = 'Sony')
```
