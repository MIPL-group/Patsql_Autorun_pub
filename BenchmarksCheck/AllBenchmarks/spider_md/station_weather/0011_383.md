# solution

```sql
SELECT t3.name,
       t3.time
FROM station AS t1
JOIN route AS t2 ON t1.id = t2.station_id
JOIN train AS t3 ON t2.train_id = t3.id
WHERE t1.local_authority = "Chiltern"
```
