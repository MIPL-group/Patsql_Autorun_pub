# solution

```sql
SELECT Roles.role_description,
       count(Employees.employee_id)
FROM ROLES
JOIN Employees ON Employees.role_code = Roles.role_code
GROUP BY Employees.role_code
HAVING count(Employees.employee_id) > 1;
```
